# 🎰 룰렛 로또 (Roulette Lotto)

로또 번호를 재미있게 뽑는 룰렛 게임 앱입니다.

## 📱 주요 기능

### 🎡 룰렛 게임
- 무지개 색상의 아름다운 룰렛으로 로또 번호 선택
- 자연스러운 회전 애니메이션과 바람 효과
- 즉시 멈추기 또는 자동 멈춤 기능
- 중복 숫자 자동 무효 처리

### 🎈 풍선 게임
- 45개의 풍선 중 6개를 터뜨려 번호 선택
- 파티클 효과와 반짝임 애니메이션
- 터진 순서에 따른 축하 메시지

### 📊 통계 대시보드
- 가장 많이 뽑힌 숫자 TOP 10
- 숫자별 출현 빈도 차트
- 최근 뽑기 히스토리
- 총 뽑기 횟수 통계

### 💾 번호 관리
- 뽑은 번호 저장 및 관리
- 날짜별 정렬된 저장 목록
- 실제 로또 공 색상으로 숫자 표시

## 🎨 특징

- **아름다운 디자인**: 무지개 색상의 룰렛과 현대적인 UI
- **부드러운 애니메이션**: 모든 인터랙션에 자연스러운 애니메이션
- **직관적인 사용법**: 누구나 쉽게 사용할 수 있는 간단한 인터페이스
- **완전 오프라인**: 인터넷 연결 없이도 모든 기능 사용 가능

## 🚀 시작하기

### 요구사항
- Flutter SDK 3.8.1 이상
- Dart SDK 3.8.1 이상

### 설치 방법

1. 저장소 클론
```bash
git clone https://github.com/chonjae2k/lotto2.git
cd lotto2
```

2. 의존성 설치
```bash
flutter pub get
```

3. 앱 실행
```bash
flutter run
```
